Proyek Arsitektur CNN

Klasifikasi
Sampah

Proyek machine learning berbasis Convolutional Neural Networks (CNN) untuk mengklasifikasikan sampah ke dalam empat kategori: Glass, Metal, Paper, dan Plastic. Unggah gambar untuk menguji model.

01 / Data & Preprocessing

Garbage Classification Dataset

Distribusi Kelas

Total 1.987 gambar dibagi ke dalam 4 kelas dengan proporsi yang dijaga menggunakan stratified split.

  • Glass 501 25.2%
  • Metal 410 20.6%
  • Paper 594 29.9%
  • Plastic 482 24.3%

Strategi Pelatihan

Penanganan class imbalance dan augmentasi data untuk memperkuat ketahanan model.

Pembagian Data

80% Train, 10% Val, 10% Test

Preprocessing

Resize 224x224 px, Normalisasi Float32

Augmentasi (Opsional)

Rotasi ±15°, Zoom, Flip, Shift, Brightness

03 / Deployment

Tech Stack & Infrastruktur

O

Format Model

Dikonversi menggunakan tf2onnx. Jauh lebih ringan (~10MB) tanpa perlu TensorFlow runtime di server.

A

Backend FastAPI

Menerima form data gambar, memproses inference dengan ONNX Runtime, dan merespons via API.

W

Frontend & Hosting

Antarmuka menggunakan Vanilla JS, Tailwind, dan GSAP. Di-host menggunakan layanan Render.com.
